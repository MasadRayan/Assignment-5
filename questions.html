Question-1: What are the different ways to select an element in the DOM?

There are many way to select an element in the dom like:
1.getElementByID(); -> it take an element by the id of the element
2.getElementsByClassName(); -> it take all the elements by the same class name
3.getElementsByTagName(); -> it take all the element by the same tag name
4.querySelector(); -> it take the only first matched element when we search by the css property
5.querySelectorAll(); -> it take all the elements when we search by the css property
6.document.forms; -> Used to select form elements by their name
7.document.links; -> return all the ancor tag element of the page
